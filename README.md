# **SpectraAnalysis**

SpectraAnalysis is a Python-based graphical software tool for processing, visualizing, and interpreting fluorescence emission data generated by spectrofluorometer platforms, particularly SpectraMax tabular output files.

The software provides an interactive interface for uploading fluorescence datasets, selecting positive and negative control wells, classifying samples, and generating spectral plots from 96-well plate experiments.

## **Features**

* Graphical user interface built with Flet  
* Support for SpectraMax tabular fluorescence files  
* Interactive 96-well plate visualization  
* Positive and negative control selection  
* Automatic sample classification based on fluorescence peak values  
* Fluorescence emission curve visualization  
* Export of generated plots as PNG images  
* Support for fluorescence-based diagnostic assays, including FRET-HCR workflows

## **Input File Format**

SpectraAnalysis was designed to process SpectraMax tabular output files. The input file should contain fluorescence emission data organized by wavelength and well position.

The expected file format includes:

* Tab-separated values;  
* UTF-16 encoding;  
* A `Wavelength` column;  
* Well columns named according to a 96-well plate format, such as `A1`, `A2`, `B1`, and `H12`;  
* Fluorescence intensity values for each wavelength;  
* Decimal values using comma notation, when applicable.

The software automatically removes the temperature column if it is present in the file.

## **Basic Workflow**

1. Open the software.  
2. Click **Upload File**.  
3. Select the SpectraMax fluorescence data file.  
4. Select the negative control wells by clicking once on each well.  
5. Select the positive control wells by clicking twice on each well.  
6. Click **Generate Results**.  
7. Inspect the classified 96-well plate map.  
8. Select individual wells, rows, columns, or all wells for plotting.  
9. Click **Plot Cells**.  
10. Review the generated fluorescence plots.

## **Control Selection**

After uploading the file, the software displays an interactive 96-well plate map.

Each valid well can be clicked to assign its role:

| Click number | Well status | Color |
| ----- | ----- | ----- |
| 0 clicks | Unassigned well | Grey |
| 1 click | Negative control | Blue |
| 2 clicks | Positive control | Red |
| 3 clicks | Reset to unassigned | Grey |

At least one positive control and one negative control must be selected before generating results.

## **Sample Classification**

SpectraAnalysis classifies samples based on fluorescence peak values relative to the negative control baseline.

The software calculates the mean signal of the negative controls and evaluates the normalized fluorescence peak of each sample in the selected wavelength interval.

In the current version, the default peak interval is:

655–665 nm

Samples above the calculated threshold are classified as positive, while samples below the threshold are classified as negative.

## **Output**

SpectraAnalysis generates:

* A classified 96-well plate map;  
* Fluorescence emission line plots;  
* Normalized peak bar plots;  
* PNG image files of generated graphs.

Generated graph files are saved automatically in the project directory using timestamped names, such as:

grafico\_1234567890.png

## **Troubleshooting**

### **The application does not start**

Make sure the virtual environment is activated and all dependencies are installed:

pip install \-r requirements.txt

Then run:

python main.py

### **`ModuleNotFoundError`**

This error indicates that one or more required libraries were not installed.

Install the missing dependencies with:

pip install flet pandas numpy matplotlib

### **The upload button works, but the file is not processed**

Check whether the input file follows the expected SpectraMax tabular format. The file should be tab-separated, encoded in UTF-16, and contain a `Wavelength` column and well columns such as `A1`, `A2`, and `H12`.

### **No valid negative controls were found**

Make sure that at least one valid well was selected as a negative control. Negative controls are required to calculate the baseline and classification threshold.

### **No data points were found between 655 and 665 nm**

The uploaded file may not contain fluorescence emission values in the wavelength range used for peak calculation. Check the wavelength range in the input file or modify the peak interval in the source code.

## **Development Notes**

The current version uses Matplotlib with the `Agg` backend to generate static PNG plots inside the Flet interface. This allows the program to render figures without requiring an interactive Matplotlib window.

The graphical interface is built with Flet, while data processing is performed using Pandas and NumPy. Matplotlib is used for spectral visualization and graph export.
